Estimation
==========

This class estimates the parameters of the Fellegi-Sunter model, a `Naive Bayes classifier <https://en.wikipedia.org/wiki/Naive_Bayes_classifier>`_ commonly employed in probabilistic record linkage, using the output generated by either the :doc:`Comparison <comparison>` or :doc:`Deduplication <deduplication>` classes.
